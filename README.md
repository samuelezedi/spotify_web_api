# spotify_web_api

A spotify web api implementation in Dart

## Basic Usage:

```dart
import 'package:mailchimp/mailchimp.dart';
```

## Spotify

### Get Root

```dart
    Spotify sp;
    sp = Spotify(
          clientID: "",
          clientSecret: "",
          redirectUrl: "http://your-domain.com/spotify-callback.html");
```
